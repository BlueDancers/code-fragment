# Koa2初始环境

koa2这个也算是最喜欢的一个node框架了,使用的比较多.总是搭建初始环境,浪费时间,这里存储一个初始化完成的koa2环境





还有一些可能会用到的

```
npm install jsonwebtoken 
npm install md5
npm install mongoose
npm install pm2
```

pm2可能用到的命令

```
"start": "pm2 start app.js --watch",
"c": "pm2 logs app [--lines 3000]",
"stop": "pm2 stop all",
"delete": "pm2 delete 0"
```
